🎤 soramimic-video 動画生成

入力

soramimicエディタで編集→「書き出し」したファイル。なしの場合は下の単語リストで自動変換します。

オプション

高すぎるときは -12(1オクターブ下)がおすすめ。

画像と列情報(説明文など)・歌詞字幕の配置。空欄はサーバー既定。

レイアウトを編集(ドラッグで配置)
- / -枚目 editor JSONの変換結果で表示中(動画のキュー順)

単語リスト欄のリストの実データ(画像・列の値)がそのまま表示され、 箱をドラッグすると中身ごと動きます。中央や他の要素に近づくとガイド線が出て吸着。 右下の角でリサイズ、クリックで選択して文言・サイズ・色を変更。 選択中はカーソルキーで移動(Shiftで大きく)、Deleteで削除。 本番と同じく、枠に収まらない文字は自動で縮小されます。 編集した内容はジョブ投入時にレイアウト名より優先で使われます。 「未知語用」タブは単語リストに行がない単語(手入力など)の配置で、 列({achievement}など)は空になるので{surface}/{original}だけで組みます (字幕は通常側と共通)。要素ごとの「表示条件」列を空にした単語ではその要素を隠せます。

JSON

プレビューは替え歌変換をせず、歌い出しから20秒ぶんを元歌詞のまま合成します。モデルや移調の当たりをつけるのに。
